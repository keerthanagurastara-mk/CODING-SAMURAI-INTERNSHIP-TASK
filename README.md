### 🔹 Project 1: Titanic Survival Prediction

- Feature engineering (FamilySize, Title)
- Preprocessing pipeline (scaling + encoding)
- Model evaluation (Accuracy, ROC-AUC, Cross-validation)

📊 Results:
- Accuracy: ~81%
- ROC-AUC: ~0.88
- Cross-validation: ~82.7%
