# 📈 Stock Price Forecasting using ARIMA & LSTM

## 🚀 Overview
This project predicts stock prices using time series forecasting techniques.

## 🔧 Models Used
- ARIMA (statistical model)
- LSTM (deep learning model)

## 📊 Features
- Time series preprocessing (ADF test, resampling)
- ARIMA forecasting
- LSTM sequence modeling
- Model evaluation (RMSE, MAE)
- 10-day future prediction

## 📈 Results
- ARIMA RMSE: ~12  
- LSTM RMSE: ~1.2  

## 🧠 Key Insight
LSTM outperforms ARIMA in capturing nonlinear patterns in stock prices.
