
# 📊 Sales Data Analysis

## 🔹 Project Overview

This project focuses on analyzing retail sales data using basic data analytics techniques. The goal is to understand sales performance, identify trends, and visualize insights using simple statistical methods.

---

## 🔹 Objectives

* Perform data cleaning and preprocessing
* Calculate basic statistics (total, average, min, max sales)
* Analyze sales by category
* Create visualizations (bar chart and pie chart)
* Derive meaningful business insights

---

## 🔹 Dataset

The dataset used is a retail sales dataset containing:

* Order Date
* Category & Sub-Category
* Product Name
* Sales
* Quantity
* Profit

---

## 🔹 Tools & Technologies

* Python
* Pandas
* Matplotlib
* Google Colab / Jupyter Notebook

---

## 🔹 Steps Performed

### 1. Data Cleaning

* Removed duplicate records
* Checked and handled missing values

### 2. Feature Engineering

* Used existing **Sales** column as Total Sales

### 3. Data Analysis

* Calculated total, average, maximum, and minimum sales
* Grouped data by category to analyze performance

### 4. Data Visualization

* Bar chart to compare sales across categories
* Pie chart to show sales distribution



## 🔹 Key Insights

* Technology category has the highest sales
* Furniture and Office Supplies contribute significantly
* Sales distribution shows clear dominance of certain categories
