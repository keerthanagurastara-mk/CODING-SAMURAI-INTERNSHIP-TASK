# Titanic Survival Prediction using Logistic Regression

## Overview
This project predicts survival on the Titanic using Logistic Regression.

## Results
- Accuracy: ~81%
- ROC-AUC: ~0.88
- Cross-validation: ~82.7%

## Features
- Age, Sex, Pclass
- FamilySize
- Title
- Fare
- Embarked

## Tech Stack
Python, Pandas, Scikit-learn
